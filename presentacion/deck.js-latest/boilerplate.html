Historia de la web para celulares

WAP - Wireless Application Protocol

  • 1989
  • Primer soporte a la web para celulares
  • En mi epoca todo era distinto...
  • Poca memoria
  • Poco cpu
  • Poco ancho de banda
  • WML (text/vmd.wap.wml)
  • WMLScript
  • WBMP
        <?xml version="1.0"?>
        <!DOCTYPE wml PUBLIC "-//WAPFORUM//DTD WML 1.2//EN"
        "http://www.wapforum.org/DTD/wml12.dtd">
        <wml>
            <card title="Bienvenido" id="inicio">
                <p><b>Hola!</b> Parece que tenes un celular viejo</p>
                <a href="#quinela">Ver numeros de la quinela</a>
                <a href="www.mercadolibre.com">Comprar un celular nuevo</a>
            </card>

            <card title="Quini 6!" id="quinela">
                <p>Los numeros de la quinela son 03 05 07 11 13 17</p>
                <a href="#inicio">Volver</a>
            </card>
        </wml>
